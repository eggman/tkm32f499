tkm32f499 is chinese mcu.

# spec

* ARM Cortex-M4 240Mhz
* 8MB SDRAM
* I2C x3 / I2S x1 / UART x5 / SPI x4 / QSPI x1 / CAN x2 / SDIO x2 / USB x1
* 128LQFP
